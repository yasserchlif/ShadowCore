# ShadowCore Discord Bot

**Overview:**  
ShadowCore is a Discord bot built in Node.js using Discord.js v14. Developed since day 1 with multiple commands and features.

**Bot Path Structure:**  
- commands/ → all bot commands  
- index.js → main bot file  
- .env.example → template for environment variables  
- .gitignore → ignores node_modules, secrets, and system files  
- messages.log → logs bot events (local)  

**Commands included:**  
- serverinfo.js  
- stats.js  
- free.js  
- free_dashboard.js  
- free_test.js  
- hello.js  
- steam_dashboard.js  
- steam_top.js  

**History & Dev Notes:**  
- Created on 2025-12-07  
- Commands added incrementally  
- Token and secrets kept safe in .env  
- Node.js project, requires npm install  

**How to run the bot:**  
1. Copy .env.example → .env and add Discord token  
2. Run npm install  
3. Run node index.js
